# Public Assets Folder

Place your static assets here. Files in this folder will be served at the root path.

## Favicon Setup

1. Place your favicon file in this folder as `favicon.ico`
2. Supported formats: `.ico`, `.png`, `.svg`
3. Recommended sizes: 16x16, 32x32, or 48x48 pixels

### Multiple Favicon Formats (Optional)

If you have multiple formats, you can add them to `index.html`:

```html
<!-- Standard favicon -->
<link rel="icon" type="image/x-icon" href="/favicon.ico" />

<!-- PNG favicon (alternative) -->
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png" />
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png" />

<!-- Apple touch icon -->
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
```
